# Employee Attrition Risk & Retention Dashboard

## Project Overview
This project analyzes employee attrition using HR analytics concepts.  
The dashboard helps HR teams understand attrition trends, identify high-risk employees, and explore key factors influencing employee exits.

## Objective
- Measure overall employee attrition
- Identify risk categories contributing to attrition
- Analyze overtime and performance impact on attrition
- Support data-driven HR decision-making

## Tools Used
- Zoho Analytics
- Excel / CSV
- HR Analytics Concepts

## Key Metrics & Visuals
- Total Attrition (KPI)
- Attrition by Risk Category
- Overtime vs Attrition
- Performance Rating vs Attrition

## Key Insights
- Majority of attrition comes from medium-risk employees
- Employees with overtime show higher attrition
- Lower performance ratings have higher attrition trends

## Outcome
This dashboard demonstrates practical application of HR analytics for workforce planning and retention strategies.
